# Digital-Literacy-Project

In this project, I worked on different tasks that helped me understand how digital tools are actually used in real student life, not just in theory.
The main idea of this project was to learn how to use technology in a smart and responsible way, and also how to present myself better online.

📌 Task 1 – Digital Literacy Infographic
For the first task, I created a one-page infographic using Canva.
I chose Canva because it was easy to use and had ready templates, so I didn’t have to start from scratch.
In the infographic, I explained what digital literacy means, some useful tools like Google Docs, Canva, YouTube, and also safe internet practices. I also added a section about online presence because I think many students ignore that part.
Honestly, the difficult part was keeping everything short but still meaningful. I had to remove some points because it was getting too crowded .
But overall, it helped me understand how to present information visually.

📌 Task 2 – Student Digital Portfolio
In this task, I created and updated my profiles on GitHub, Kaggle, and LinkedIn.
On GitHub, I uploaded small projects like a calculator and a password generator. It helped me see my progress and keep things organized.
On Kaggle, I practiced Python using small exercises. Sometimes I got stuck, but retrying actually helped more.
LinkedIn was different — I used it to build a professional profile and add my education details.
I feel this task is useful in the long run because these platforms will matter a lot for internships and future work.

📌 Task 3 – Coding & Collaboration Platforms
Part A – Coding Practice
I used HackerRank for coding practice and completed beginner-level Python problems. I also earned a Python badge.
At first, I didn’t get some answers correct and had to try again and again, which was a bit frustrating . But that’s where I actually learned more.
Part B – Google Form
I created a Google Form quiz based on digital literacy.
The quiz includes 5 questions covering safe internet use, digital tools, and online presence. I used different types of questions like multiple choice, checkboxes, and one short answer.
One thing I liked is that responses are automatically saved and easy to view. It made me realize that tools like this are actually very useful for surveys and feedback also.

📌 Task 4 – Email & Communication
In this task, I wrote two professional emails — one for requesting an extension and one for internship inquiry.
I also created a checklist for social media do’s and don’ts.
One thing I noticed is that many students (including me sometimes ) send very short or unclear messages like “send details” which can sound rude even if we don’t mean it that way.
After this task, I understood that even small things like proper greeting and explanation matter a lot in communication.

📌 Task 5 – Cybercrime Awareness
For this task, I worked on a cybercrime case study (UPI fraud).
What surprised me the most is that these scams are actually very simple. They don’t need hacking skills — they just trick people into making quick decisions.
I also made a prevention checklist with tips like avoiding unknown links, checking payment requests, and using strong passwords.
One habit I will change is being more careful before clicking anything or accepting payment requests. Earlier I didn’t think much, but now I double-check.
