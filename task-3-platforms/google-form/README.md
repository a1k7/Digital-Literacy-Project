For this part of the task, I created a Google Form to design a small Digital Literacy Awareness Quiz for my classmates. The idea was to keep it simple but still cover some important basics that we often ignore while using the internet daily.


The form contains exactly 5 questions. I included different types of questions like multiple choice, checkboxes, and one short answer question. For example, there is a question about safe internet practices, one about useful digital tools, and another about why online presence matters. I tried to keep the questions easy to understand so that anyone can attempt it without confusion.

While making the form, I also explored features like adding options, marking correct answers, and checking the response section. One thing I found useful is that all responses are automatically saved and can be viewed in a linked Google Sheet, which makes it easy to analyze answers later.

Honestly, at first it felt like a very basic task, but while creating the questions I realized it's not that easy to frame them clearly. You have to think from the user's side.
